Project Overview

This project analyzes customer churn behavior using Power BI. The dashboard provides insights into customer retention, churn rate, and key factors influencing churn.

🎯 Key Insights

Total Customers: 6418

Churn Rate: 27%

Highest churn observed in Month-to-Month contracts

Customers with <6 months tenure show higher churn

Payment method and state also impact churn

📊 Dashboard Features

Churn Rate by Gender

Churn by Age Group

Churn by State

Churn by Payment Method

Churn by Contract Type

Churn by Tenure Group

🛠 Tools Used

Power BI

Excel / CSV Dataset

Data Cleaning using Power Query
